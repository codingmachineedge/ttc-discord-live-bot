# Feature Docs

This folder breaks the bot's user-facing features into one Markdown file per feature area.

## Files

- [commands.md](commands.md) - Slash commands and message-triggered command behavior.
- [service-alerts.md](service-alerts.md) - TTC alerts, TTC.ca status checks, alert graphics, routing, pings, and cleanup.
- [vehicle-lookup.md](vehicle-lookup.md) - Vehicle list output and realtime source behavior.
- [line-5-realtime.md](line-5-realtime.md) - Line 5 realtime fallback behavior and data fields.
- [line-5-departure-board.md](line-5-departure-board.md) - Thread-based live Line 5 departure boards.
- [trip-follower.md](trip-follower.md) - Vehicle-number trip following, next-stop announcements, and graphics.
- [trip-recommendations.md](trip-recommendations.md) - Eglinton eastbound recommendation logic and GIF output.
- [station-details.md](station-details.md) - Door side, accessibility, washroom, and depth details.
- [general-feedback.md](general-feedback.md) - General-channel feedback acknowledgement.
- [manual-live-test.md](manual-live-test.md) - Manual live-test bot behavior.
