# Station Details

Trip follower messages include station detail lines where data is available.

Details can include:

- Door opening side.
- Elevator status.
- Escalator status.
- Washroom availability and status.
- Station depth.
- Notes.

Some values are configured manually in code. Some accessibility and amenity statuses may be inferred from active TTC alerts. Unknown values are reported as unknown rather than invented.

Line 5 door side details are best effort. Surface station behavior is handled separately where known.
